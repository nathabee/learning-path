Wordpress

Overview

What is WordPress?

WordPress is a free and open-source content management system (CMS) based on PHP and MySQL. It features a plugin architecture and a template system, referred to as Themes. WordPress is used by over 40% of all websites on the internet.

Why Use WordPress?

WordPress is user-friendly, flexible, and scalable. It offers a range of features including a wide variety of themes and plugins, making it easy to create any type of website, from blogs to e-commerce stores.

Key Features

  • User-friendly interface
  • Extensive plugin and theme ecosystem
  • SEO-friendly
  • Responsive design
  • Strong community support

Installation

Installation on Ubuntu

# Update your package index
sudo apt update

# Install necessary packages
sudo apt install apache2 mysql-server php php-mysql libapache2-mod-php php-cli

# Download WordPress
wget -c http://wordpress.org/latest.tar.gz

# Extract WordPress
tar -xzvf latest.tar.gz

# Move WordPress to the Apache directory
sudo mv wordpress/* /var/www/html/

# Set permissions
sudo chown -R www-data:www-data /var/www/html/
sudo chmod -R 755 /var/www/html/

# Create WordPress configuration file
sudo cp /var/www/html/wp-config-sample.php /var/www/html/wp-config.php

# Open MySQL and create a database
sudo mysql -u root -p
CREATE DATABASE wordpress;
GRANT ALL PRIVILEGES ON wordpress.* TO 'wordpressuser'@'localhost' IDENTIFIED BY 'password';
FLUSH PRIVILEGES;
EXIT;

# Configure wp-config.php file
sudo nano /var/www/html/wp-config.php
// Add the database details and save the file

# Restart Apache
sudo systemctl restart apache2

Apache Configuration

# Enable Apache mod_rewrite
sudo a2enmod rewrite

# Configure Apache to allow .htaccess overrides
sudo nano /etc/apache2/sites-available/000-default.conf
# Add the following within the  directive:
# AllowOverride All

# Restart Apache
sudo systemctl restart apache2

Important Modifications Since 2018

WordPress 5.0 (2018)

Introduction of the new block editor, Gutenberg. This update changed the way content is created and edited, moving from the classic editor to a block-based editor.

WordPress 5.5 (2020)

Enhanced the block editor with new features, improved media handling, and introduced lazy loading for images and iframes.

WordPress 5.6 (2020)

Further improvements to the block editor with additional blocks and patterns, along with support for WebP images.

WordPress 5.9 (2022)

Introduced full site editing capabilities, allowing users to edit headers, footers, and other site-wide elements using the block editor.

Introduction to Blocks

What are Blocks?

Blocks are the building blocks of content in WordPress. They allow you to create and organize content in a modular way. Each block represents a different piece of content, such as text, images, videos, or quotes.

Using the Block Editor

The block editor, also known as Gutenberg, provides a visual interface for creating and editing content. You can add, rearrange, and style blocks to create complex layouts with ease.

Common Blocks

  • Paragraph
  • Image
  • Heading
  • Gallery
  • List
  • Quote
  • Audio
  • Cover
  • File
  • Video

Reusable Blocks

Reusable blocks allow you to create content that can be used across multiple posts and pages. This feature is especially useful for content that you want to reuse, such as a call to action or a signature block.

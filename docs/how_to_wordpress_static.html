Convert WordPress Site to Static HTML

Using WP2Static

To convert your WordPress site to static HTML, you can use plugins such as WP2Static or Simply Static.

Install WP2Static

  1. Go to your WordPress admin dashboard.
  2. Navigate to Plugins > Add New.
  3. Search for "WP2Static" and install it.

Configure WP2Static

After activation, go to the WP2Static settings and configure the settings according to your needs. You can specify where the static files will be saved and other options.

Generate Static Files

Once configured, you can generate the static files by clicking the "Generate" button and download the generated static files.

Host on GitHub Pages

Create a GitHub Repository

Go to GitHub and create a new repository named yourusername.github.io.

Upload Static Files

  1. Clone the repository to your local machine using git clone https://github.com/yourusername/yourusername.github.io.git.
  2. Copy the static files generated by WP2Static to the repository directory.
  3. Commit and push the changes: 
    
    cd yourusername.github.io
    git add .
    git commit -m "Initial commit of static site"
    git push origin main

Access Your Site

Your static site should now be accessible at https://yourusername.github.io.

Additional Tips

Custom Domains

If you want to use a custom domain, you can set this up in the GitHub repository settings under the "Pages" section.

Updates

Whenever you make changes to your WordPress site, you'll need to regenerate the static files and push them to GitHub again.

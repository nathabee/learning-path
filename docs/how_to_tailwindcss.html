Overview

Tailwind CSS is a utility-first CSS framework for building custom designs without writing traditional CSS. It provides low-level utility classes that let you compose complex designs directly in your HTML.

Key features include:

  • Utility-first approach
  • Highly customizable with configuration
  • Responsive design support

Installation

To install Tailwind CSS, you can use npm or yarn:

# Using npm
npm install tailwindcss

# Using yarn
yarn add tailwindcss

Configuration

After installing Tailwind CSS, you need to configure it:

# Create a Tailwind configuration file (if not already present)
npx tailwindcss init

This command generates a `tailwind.config.js` file where you can customize various aspects of Tailwind CSS, such as colors, fonts, breakpoints, etc.

Tailwind CSS Configuration

To customize Tailwind CSS, you can modify the `tailwind.config.js` file:

# Create a Tailwind configuration file
npx tailwindcss init

This command generates a `tailwind.config.js` file where you can customize various aspects of Tailwind CSS, such as colors, fonts, breakpoints, etc.

Example of `tailwind.config.js`:

module.exports = {
  purge: [],
  darkMode: false, // or 'media' or 'class'
  theme: {
    extend: {
      colors: {
        'primary': {
          'light': '#8fc7c3',
          'DEFAULT': '#00796b',
        },
        'secondary': {
          'light': '#fadea1',
          'DEFAULT': '#928974',
        },
        'tertiary': {
          'light': '#eba5a5',
          'DEFAULT': '#700606',
        },
        'highlight': '#ffff00',
      },
    },
  },
  variants: {
    extend: {},
  },
  plugins: [],
}

Usage with tailwind classes called in html

Once installed and configured, you can start using Tailwind CSS classes in your HTML:

<div class="bg-blue-500 text-white p-4">
    <p class="font-bold">Tailwind CSS Example</p>
    <p>Easy to use and customize.</p>
</div>

This example applies a blue background, white text, padding, and bold font to a div element.

example of a tailwind.css

/* docs/tailwind.css */
                        @tailwind base;
                        @tailwind components;
                        @tailwind utilities;
                        
                        @layer base {
                          :root {
                            @layer base {
                              :root {
                                --primary-color-light: #6CB2B8;
                                /* Light teal */
                                --primary-color: #2C7A7B;
                                /* Teal */ 
                              }
                        
                            }
                        
                          }
                        
                          h1 {
                            @apply text-3xl font-bold mb-4;
                          }
                        
                          h2 {
                            @apply -xl font-semibold mb-2 border-b-2text pb-2 bg-tertiary-color-light border-primary-color;
                          }
                        
                          h3 {
                            @apply text-lg font-medium;
                          }

This example applies a blue background, white text, padding, and bold font to a div element.
